# 🎀 Moefun

个人博客，基于 [Astro 6](https://astro.build) 构建，主题改编自 [Fuwari](https://github.com/saicaca/fuwari)。

## ✨ 特性

- 🚀 基于 Astro 6.1 + Content Layer API
- 🎨 亮色 / 暗色主题切换
- 📱 响应式设计
- 🔍 内置搜索 (Pagefind)
- 📝 Markdown 扩展语法支持
- 📊 目录导航 (TOC)
- 📡 RSS 订阅

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 📝 新建文章

```bash
pnpm new-post <filename>
```

文章放在 `src/content/posts/` 目录下。

## 📄 配置

编辑 `src/config.ts` 自定义博客的标题、头像、Banner 等设置。
